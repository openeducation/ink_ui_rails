# InkUiRails

InK for Rails Asset Pipleline

Need help for InK itself. Go to the [InK homeapge](http://ink.sapo.pt/)

## Installation

Add this line to your application's Gemfile:

    gem 'ink_ui_rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ink_ui_rails
